# README

The Friends Toolkit project demonstrates a basic implementation of the Friends service.

## Getting started

Before continuing, [download](https://unity3d.com/get-unity/download) and install the [Unity Hub](https://unity3d.com/get-unity/download/) and a supported version of the Unity Editor. The Friends service supports the following Editor versions:

- Version 2021 or later
- Version 2020.3 or later

### Download the sample project

Download the Friends sample project and open it in a supported version of the Unity Editor. You’ll want to have the sample project open while you [enable the Friends service](https://docs.google.com/document/d/1tP-Zl8aWZ-vVGHecMo-mD7-T_snkQESrWHyFi2MAXlc/edit#heading=h.1s70w5qyvm6l).

### Enable the Friends service

1. Log into the Unity Dashboard. If you don’t already have an account, see [Get started with Unity Gaming Services (UGS)](https://docs.unity.com/ugs-overview/GettingStarted.html).
2. Go to **Multiplayer** > **Friends**.

![](https://lh4.googleusercontent.com/lh4cH-632w4X0THy1tIexPRLT79oTbfeDQRxi6sqGLtp6-ESuUgO4OylxdI3FhJrwLL_7tVRxy5JQvh5UFRhIBBz5Z1frdSstAMpV6eZ6_YGKykRr8q0_x6y7vK8_MR89-0c4kRtfXtTshrr4dOZum5ontjfU4DhPWRCRtwCuoX49-ddKTa4OvaCTGzQlQ)

3. Select **Set up Friends**.

![](https://lh4.googleusercontent.com/aIgEQM1qD_hGGEMbiIUKyMbsF7IzEAqWYDmD16aahfx8_3IRVl7g1Gys2zvFhHQyFRcr7e7Ebz0fS0QDnwqMVgWRnBE4KpsWyFttOU0RQCxayEvBu8Fi4sYdQvrIGcJzkTD_UsIUQW1hLb4WvTUj-e8wVwN127hMW2pu0gcQiYCyWsdSwXcwVwPHqXprXw)

4. If you haven’t already added your payment information, the Unity Dashboard might prompt you to add your payment information. Friends is a [pay as you go service with a free tier](https://unity.com/solutions/gaming-services/pricing). However, Friends is completely free during the beta release.

![](https://lh3.googleusercontent.com/m42jSrFXdsJ2IWdTi4zgMRLfia3VvfUDJSL1CbhqG9N5DyEU-jT1TbPNSv43KfHkCZ3ZH42pW5x3kAL-cvXnHUksD3fbLsg-3PfVkiznyjOUGnuqWT6bBGrKJbEHjMDGu4BhI0TNey2cvZuJQJYrPiJw13_L2LEXuARycjqmE65_TCkDWaIVPO-YoACgrA)

5. Continue onto the Setup guide, which walks you through setting up your Friends project.

#### Link your Friends project with the Unity Editor

**Note**: You can safely skip this step if you've already linked your project through the Unity Engine. You can also link your project after enabling the Friends service.

1. Follow the instructions at [Link your project in the Unity Editor](https://docs.unity.com/ugs-overview/GettingStarted.html#LinkProject).
2. Select **Next** in the Setup guide.

#### Install the Friends package

1. You can use Unity Editor version **2021 or later** or version **2020.3 or later**. Select the version you want to use in the Setup guide on the Unity Dashboard, then follow the instructions.
2. When you’re finished, select **Next** in the Setup guide.

#### Turn Friends on

1. Activate Friends by toggling the in the Setup guide to **On**. Once activated, players can send friend request to other players and build a friends list.

![](https://lh6.googleusercontent.com/DSiwMDslsBRystTnFLpsg01HxOLOTspw7_eaBduq9NHsG83AulqAn3Vvt3kkw86zl3CtoziypqsCiF77xpu8UB9RuRqvV7_HwYi1mB31BSgJAINsBQhOfQJdnV_SzD3Rua4lqYHgYiijrw1AE2RNKGfoXS5q0TUFc9Oua4fGi-GKknTiTN17Om6R5lDfQw)

2. Select **Finish**.

**Note**: If you encounter any conflicts with other services, deactivate the Friends service. You can turn Friends on or off at any time.

## Play with the sample project

At this point you should have the sample project open in the Unity Editor. It’s time to interact with the sample.

**Note**: If you haven’t already, log into the Unity Editor and [link your Unity project](https://docs.unity.com/ugs-overview/GettingStarted.html#LinkProject) before continuing.

The sample project consists of a simple user interface that allows you to interact with the Friends service and perform actions such as:

- Send a friend request
- Accept a friend request
- Ignore a friend request
- Block a player
- View your Friend list
- View your Block list
- Unblock a player
- Remove a friend
- Set your Presence status

It also includes a debugging tool that allows you to:

- Refresh the user interface

You can set your Presence status by selecting the down arrow next to your current status, then selecting the new status. For example, to change your status from ONLINE to OFFLINE:

1. Select the down arrow next to ONLINE.
2. Select **OFFLINE**.
